# Запуск і перевірка GitШлях

Це Next.js 16 (Turbopack) застосунок. «Запустити» = підняти dev-сервер і **проїхати потоком як користувач**, а не лише зібрати типи.

## Підняти сервер

```bash
npm run dev        # Next dev, за замовчуванням http://localhost:3000
```

- Порт може зайняти інший інстанс — Next сам візьме `3001` і напише про це в лог. Перевір рядок `Local: http://localhost:PORT`.
- Якщо вже висить попередній dev-сервер (Next скаже «Another next dev server is already running» з PID) — використовуй наявний `localhost:3000`, не плоди дублікати. Зупинити зайвий: `taskkill /PID <pid> /F`.
- Готовність: чекай на `✓ Ready` у логах перед тим, як бити по сервері.

Швидка перевірка живучості:
```bash
curl -s -o /dev/null -w "%{http_code}\n" http://localhost:3000
```

## Проїхати потоком (як користувач)

Застосунок — SPA зі станом-екранами всередині `GitWayApp.tsx`. Реального роутінгу немає; переходи — кнопками. Ключові місця для перевірки правок:

- **login → roadmap**: обери акаунт (напр. тестовий) → зʼявляється дорожня карта.
- **lesson**: відкрий урок з карти → відео/аудіо-слоти, теорія, аналогія, кнопка практики, «Пройти квіз».
- **quiz**: пройди питання → екран «Урок завершено!» з кнопками «Наступний урок», «Далі до карти», «Мій прогрес» (hover — підйом кнопки).
- **sandbox**: справжній Git-термінал + GitHub-клон; термінал має заповнювати всю область, нижня панель підказок (Tab/↑↓/Ctrl+L/Ctrl+C) видима.
- **cli**: симулятор Claude Code / Codex у тому ж компоненті Terminal.

## Драйв через браузер

Якщо доступні браузерні інструменти (розширення Claude in Chrome / chromium-cli) — відкрий `localhost:3000`, клікни через потрібний екран і **зроби скріншот**; порожній кадр = провал запуску. Якщо браузерні інструменти вимкнено в сесії — попроси користувача глянути екран вручну на `localhost:PORT` і опиши, що саме перевірити.

## Швидша перевірка без браузера

- Типи: `npx tsc --noEmit`
- Тести (контент, git-рушій, компоненти): `npm test`
- Лінт: `npm run lint`

Ці перевірки не замінюють візуальний прогін для UI-правок, але ловлять регресії дешево.
