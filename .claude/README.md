# Harness проєкту GitШлях (`.claude/`)

Обвязка для автоматизації типової роботи над навчальним застосунком: контент уроків, відео, UI та запуск/перевірка. Скіли й агенти закомічені в репозиторій — тож доступні всій команді.

## Скіли (`.claude/skills/`)

| Скіл | Коли |
|------|------|
| **write-lesson** | Додати/редагувати урок CLI-курсу (JSON-контент Claude/Codex) за схемою з валідацією. |
| **render-lesson-video** | Відрендерити відео-урок через Remotion у `public/videos` (id 2–35), студія прев'ю, обхід пастки з кирилицею в шляху. |
| **run-dev** | Підняти Next dev-сервер і проїхати потоком (login → lesson → quiz → sandbox/cli) для живої перевірки правок. |

Виклик: `/write-lesson`, `/render-lesson-video`, `/run-dev` (або автоматично за описом задачі).

## Агенти (`.claude/agents/`)

| Агент | Роль |
|-------|------|
| **lesson-author** | Пише україномовний контент уроків за схемою й тоном проєкту (теорія, аналогія, пісочниця, командний квіз). |
| **clay-ui** | Будує UI у фірмовій claymorphism-ідіомі (`sx()` + `Clay` + `Icon`), без сторонніх бібліотек. |

Виклик через Task/Agent із відповідним `subagent_type`.

## Джерела істини

- `src/content/README.md` — схема контенту уроків і каталогу команд.
- `remotion/README.md` — рендер відео.
- `AGENTS.md` — правила Next.js цієї версії (читати доки в `node_modules/next/dist/docs/`).
